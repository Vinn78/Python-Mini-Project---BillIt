## What is BILLIT?

BILLIT is a simple billing app made with Python using Tkinter for the user interface. It lets you add products, enter their quantity and price, and see the total amount for each product directly in the app. This app is easy to use and great for beginners who want to learn Python and GUI programming.

## What Can It Do?

* Let you enter product name, quantity, and price.
* Automatically calculate the total price for each item.
* Display all product details directly in the app without popups.
* Show an error message if the quantity or price is not entered correctly.

## How It Looks

The app has a simple window with a box to type the product name, a box for quantity, a box for price, and a button to add the item. The total is shown directly in the app interface.

## How to Run It

1. Make sure Python 3 is installed on your computer.
2. Download the project files.
3. Open the folder and run:

   ```
   python billit.py
   ```
4. Enter the product details and click **Add Item to Bill** to see the total displayed in the app.
